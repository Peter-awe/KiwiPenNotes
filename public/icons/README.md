# PWA Icons — Placeholder

The following icon files are required for the PWA manifest and Apple touch icon:

- **icon-192.png** — 192x192 px, PNG (used by Android home screen + Apple touch icon)
- **icon-512.png** — 512x512 px, PNG (used by Android splash screen)

Both icons should use the KiwiPenNotes branding (kiwi bird / pen motif).
You can generate them from one of the SVG concepts already in this folder.

Requirements:
- Square, no transparency for maskable icon areas
- Safe zone: keep the main graphic within the center 80% for maskable icons
