## 🚀 Getting Started (Development) | 开发指南

### Prerequisites / 前置要求

- **Node.js 20+** (LTS recommended / 推荐 LTS 版本)
- **npm** or **pnpm**
- A Supabase project (for auth & database)
- A Stripe account (for payments, optional for dev)

### Setup / 安装

```bash
# 1. Clone the repository / 克隆仓库
git clone https://github.com/Peter-awe/KiwiPenNotes.git
cd KiwiPenNotes

# 2. Install dependencies / 安装依赖
npm install

# 3. Configure environment / 配置环境变量
cp .env.example .env.local
# Edit .env.local with your Supabase & Stripe keys
# 编辑 .env.local 填入你的 Supabase 和 Stripe 密钥

# 4. Start development server / 启动开发服务器
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

### Deploy to Cloudflare Workers / 部署到 Cloudflare Workers

```bash
# Build with OpenNext for Cloudflare
npm run cf:build

# Preview locally
npm run cf:preview

# Deploy to production
npm run cf:deploy
```

> CI/CD is configured — every push to `main` triggers automatic deployment via GitHub Actions.
>
> CI/CD 已配置——每次推送到 `main` 分支会自动触发 GitHub Actions 部署。
